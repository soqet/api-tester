# Simple api tester

This application parses json file with requests 
and sends them.  
After all prints result in console.  


## How to install

```
git clone https://github.com/soqet/api-tester.git
cd api-tester
make build
cd builds
```
On windows:
```
./main.exe help test
```
On other systems:
```
./main.out help test
```
